Shopping Car, Carrito de compras con registro de usuario en mysql, hecho con Jade, node.js y mongoDB

¿Qué es esto?

Es mi primera aproximación creando aplicaciones web en Node.js utilizando como base de datos MongoDB, enfocada en un ShoppingCar

¿Qué necesito?

Debes tener instalado Node.js y MongoDB para poder correr la aplicacion

¿Cómo ejecuto la aplicacion?

  1. Inicializas tu servidor MongoDB (super usuario) $ mongod
  2. Corres tu aplicacion: node app.js
  3. Abres en tu navegador localhost:3000
