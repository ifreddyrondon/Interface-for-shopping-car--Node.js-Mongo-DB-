Shopping Car, Carrito de compras con registro de usuario en mysql, hecho con Jade, node.js y mongoDB

¿Qué es esto?

Es mi primera aproximación creando aplicaciones web en Node.js utilizando como base de datos Mysql para el registro de usuarios y MongoDB para los modelos, enfocada en un ShoppingCar

¿Qué necesito?

Debes tener instalado Node.js, Mysql, MongoDB para poder correr la aplicación

¿Cómo ejecuto la aplicación?

  1. Cargar la base de datos de MongoDB dentro de la carpeta kioskoBDBackup/kiosko/ en tu manejador
  2. Inicializar el servidor MongoDB (super usuario) $ mongod
  3. Inicializar el servidor Mysql
  4. Cargar la base de datos dentro de la carpeta kioskoBDBackup/kiosco.sql en tu manejador
  5. Corres tu aplicación: node app.js
  6. Abres en tu navegador localhost:3000
